# 太空弹幕射击游戏设计文档

## 项目概述

一个基于 HTML5 Canvas 的纵向卷轴弹幕射击游戏，融合 Roguelite 元素和永久升级系统。

---

## 核心玩法

### 基础操作
- **移动**: 鼠标控制自机位置
- **射击**: 按住左键自动连发
- **道具**: 靠近自动吸附，点击可主动拾取

### 战斗机制
| 元素 | 说明 |
|:---|:---|
| 连击系统 | 2秒内连续击杀维持连击，高连击有分数加成 |
| 僚机系统 | 最多3架僚机，呈三角阵型跟随射击 |
| 闪避判定 | 无hitbox显示，依赖玩家走位 |
| 时间缓速 | 拾取道具触发0.4倍速，持续10秒 |

## 敌人设计

### 普通敌人类型

| 类型 | 外观 | HP | 速度 | 特性 |
|:---|:---|:---:|:---:|:---|
| basic | 红色方块脸 | 2 | 1.5 | 最基础的敌人 |
| fast | 橙色三角 | 1 | 2.5 | 速度快但脆弱 |
| tank | 灰色坦克 | 5 | 0.8 | 重型装甲，血厚 |
| shooter | 紫色圆球 | 2 | 1.2 | 会发射追踪弹 |
| splitter | 青色菱形 | 3 | 1.0 | 死亡分裂成3个小怪 |

### Boss系统

#### 基础属性（Lv.0）
- **体型**: 80px
- **血量**: 90（3管×30）
- **速度**: vx=2, vy=0.3
- **弹幕间隔**: 1500ms

#### 成长机制
每击杀1个Boss，新Boss获得强化：
| 属性 | 成长值 |
|:---|:---:|
| 体型 | +5 |
| 血量 | +30 |
| 移速 | +0.3 vx, +0.1 vy |
| 弹幕数量 | +1 |
| 弹幕间隔 | -150ms（最低500ms） |

#### 弹幕模式（随阶段解锁）
1. **散射** - 扇形扩散弹
2. **追踪弹** - 瞄准玩家射击
3. **环形** - 360度全屏弹幕
4. **交叉弹幕** - 双炮管交叉射击（阶段2+）
5. **全屏弹幕** - 高密度环形+追踪（阶段1）

#### 血条设计（DNF风格）
- 单一长条，黑色边框
- 颜色随阶段变化：绿→黄→红
- 显示当前/最大血量数值

## 数值设计

### 玩家成长曲线
```
初始状态:
- 射速: 150ms/发
- 子弹大小: 1.0
- 伤害: 1
- 移速: 正常

理想毕业（约30分钟）:
- 射速: 50ms/发
- 子弹大小: 3.0
- 伤害: 10+
- 僚机: 3架
```

### 难度曲线
- **出怪间隔**: 初始800ms → 最低150ms
- **击杀加速**: 每10击杀，递减速度+1
- **Boss加速**: 每击杀1个Boss，额外+2递减

### 分数系统
- 基础击杀: 10分 × 连击倍率
- Boss击杀: 500分
- 连击倍率: 1.5x / 2x / 3x / 5x

## 技术实现

### 核心架构
```
game.js
├── 玩家系统（Player）
├── 子弹系统（Bullet）- 含追踪逻辑
├── 敌人系统（Enemy）- 5种类型+Boss
├── 僚机系统（Wingman）- 三角阵型
├── 道具系统（Item）- 20+种
├── 粒子系统（Particle）
└── Buff系统 - 时间累加机制
```

### 关键变量
```javascript
// 生成控制
let enemySpawnInterval = 800;  // 初始出怪间隔
let bossKillCount = 0;         // Boss击杀计数（难度成长）
let killCount = 0;             // 总击杀数

// 玩家属性
let playerStats = {
    fireRate: 150,             // 射速
    bulletSize: 1,             // 子弹大小
    bulletSizeBuff: 1,         // 巨弹倍率
    damage: 1,                 // 伤害
    multiShot: 1,              // 散射数量
    wingmanCount: 0,           // 僚机数量
};
```

### 性能优化
- 对象池：子弹/粒子超出屏幕后标记inactive
- 碰撞检测：AABB简化检测
- 渲染：Canvas 2D，无WebGL
